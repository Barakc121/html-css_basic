# מדריך עזר ל-HTML: מתי משתמשים בכל תגית?

סיכום מרוכז של התגיות הנפוצות ביותר ותפקידן הסמנטי באתר.

---

## 1. מבנה הדף (Layout)
תגיות אלו מגדירות את האזורים הגדולים באתר ועוזרות לגוגל להבין את המבנה.

| תגית | שם רשמי | מתי להשתמש? |
| :--- | :--- | :--- |
| `<header>` | Header | ראש האתר: לוגו, שם האתר ותפריט ראשי. |
| `<nav>` | Navigation | תפריטי ניווט בלבד (קישורים לדפים אחרים). |
| `<main>` | Main Content | התוכן המרכזי והייחודי של הדף (מופיע פעם אחת בלבד). |
| `<section>` | Section | קבוצת תוכן סביב נושא מסוים (למשל: "אודות"). |
| `<article>` | Article | תוכן עצמאי (פוסט בבלוג, כתבה, תגובה). |
| `<aside>` | Aside | סרגל צד: תוכן משני (פרסומות, קישורים מומלצים). |
| `<footer>` | Footer | תחתית האתר: זכויות יוצרים, פרטי קשר. |

---

## 2. טקסט ותוכן
תגיות אלו משמשות להצגת המידע הכתוב בדף.

| תגית | מתי להשתמש? | הערה חשובה |
| :--- | :--- | :--- |
| `<h1>` | כותרת ראשית | הנושא הכי חשוב בדף. משתמשים ב-h1 אחד בלבד! |
| `<h2>` - `<h6>` | תתי-כותרות | סדר יורד של חשיבות (h2 הוא תת-נושא של h1). |
| `<p>` | פסקה | לכל טקסט רגיל וסיפורי. |
| `<strong>` | הדגשה חזקה | למילים חשובות במיוחד (יוצג כ-Bold). |
| `<em>` | דגש | להדגשת נימת דיבור או מונח (יוצג כ-Italic). |

---

## 3. רשימות (Lists)
לארגון פריטים אחד אחרי השני.

* `<ul>` - רשימה עם בולטים (נקודות).
* `<ol>` - רשימה ממוספרת (1, 2, 3...).
* `<li>` - פריט בתוך הרשימה (חייב להיות בתוך `ul` או `ol`).

---

## 4. מדיה ואינטראקציה
הפיכת האתר לדינמי וויזואלי.

| תגית | תכונות חובה (Attributes) | תפקיד |
| :--- | :--- | :--- |
| `<a>` | `href` | קישור לדף אחר או אתר חיצוני. |
| `<img>` | `src`, `alt` | הצגת תמונה (alt הוא תיאור טקסטואלי). |
| `<button>` | - | כפתור לביצוע פעולות (כמו פתיחת חלון). |

---

## 5. קופסאות ועיצוב (Generic)
כשאין תגית סמנטית שמתאימה וצריך קופסה רק בשביל ה-CSS.

* `<div>` - קופסה מסוג **Block** (תופסת שורה שלמה).
* `<span>` - קופסה מסוג **Inline** (לא שוברת שורה, לעיצוב מילה בתוך משפט).

---

## חוקי ברזל לסדר בקוד
1. **סמנטיקה:** תמיד תעדיף תגית עם משמעות (כמו `<nav>`) על פני `<div>` סתמי.
2. **קינון (Indentation):** תגית שבתוך תגית אחרת תמיד תקבל "Tab" ימינה.
3. **סגירה:** כל תגית שפתחת `< >` חייבת להיסגר `</ >` (חוץ מ-`<img>`, `<br>`, ו-`<hr>`).
